Este projeto é uma implementação dos testes para efetuar login no portal de sistemas da Segplan, utilizando as seguintes ferramentas/frameworks:
1 - Padrão de testes Page Objects, para organização das classes de teste.
2 - Selenium Webdriver - para automatização das classes de teste.
3 - Spring (para obtenção do driver do selenium, por injeção de dependência).
4 - Cucumber (para implementação do teste baseado em comportamento).
